# 📚 Library Management System

A menu-driven Library Management System developed using Python and Object-Oriented Programming principles. The project allows librarians to manage books and members, issue and return books, and search the library catalog through a simple command-line interface.

---

## 🚀 Features

* Add new books to the library catalog
* View all available books
* Remove books from the system
* Register new library members
* View registered members
* Issue books to members
* Return issued books
* Search books by title
* Track issued and available books

---

## 🏗 Class Structure

### Book Class

Stores information about each book including:

* Book ID
* Title
* Author
* Issue status
* Member currently holding the book

### Member Class

Stores information related to library members including:

* Member ID
* Name
* Borrowed books list

### Library Class

Acts as the central management system and handles:

* Book management
* Member management
* Book issue and return operations
* Search functionality

## ▶️ How to Run

Clone the repository:

```bash
git clone https://github.com/your-username/Library-Management-System.git
```

Navigate to the project folder:

```bash
cd Library-Management-System
```

Run the program:

```bash
python library_management.py
```
